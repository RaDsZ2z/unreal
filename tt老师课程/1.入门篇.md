# 9.角色入门

## 9.67.替换角色的骨骼网格体

这几个小节演示了如何使用新的骨骼动画，版本问题，视频里老师讲的东西都不可用了(我的虚幻引擎版本是`5.2.1`)  

但我还是摸索出来了如何替换，步骤如下  

+ `LearningKit_Games/Assets/Characters/Character/Mesh`路径下的“骨骼网格体”和“骨骼”  复制到自己创建的文件夹中并改名

> 我的路径是`_STUDY/Characters/Orion/Mesh` 改名为`OrionCharacter`

+ 右键点击“骨骼 	网格体”，依次选择 骨骼 -> 指定骨骼 然后选择一起复制过来的那个骨骼
+ 进入当前使用人物的编辑界面，网格体->骨骼网格体 选择上面复制过来的骨骼网格体

> 现在主角已经替换成新的模型了，但是奔跑跳跃什么的动作还是旧的，接下来试着把另外任意一个奔跑跳跃的动作应用到当前角色上

+ 进入`Characters/Mannequins/Animations`路径，右键点击`ABP_Manny` -> 重定向动画资产 -> 复制并重定向动画资产/蓝图

> 发现这里右键点击了哪个文件不重要，重要的是点击的这个文件要有“重定向动画资产”这个选项

+ “IK重定向器”选择第一个，“源骨骼网格体”选择`SK_EpicCharacter_Optimized`“目标骨骼网格体”选择`OrionCharacter`文件夹选择到自己的`Animation`点击重定向

![pic18](./pngs/pic18.png)

+ 然后进到自己的`Animation`，将动画蓝图文件改个名(如`Orion_AnimBP`)
+ 然后编辑正在使用的人物，在 动画->动画类 中选择刚刚改名的`Orion_AnimBP_C`(`_C`后缀应该是引擎自己加上去的)

现在就能使用这套新的骨骼动画了，有新的奔跑跳跃和其它效果。
